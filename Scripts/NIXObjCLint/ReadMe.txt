    Инструкция конфигурирования NIXObjCLint.

    NIXObjCLint директория должна быть добавлена в дерево проекта следующим образом:

Source/
    ...
Scripts/
    NIXObjCLint/
    ...
<project name>.xcodeproj

    NIXObjCLint.py скрипт должен выполняться при каждой сборке, соответственно, нужно добавить его в "Build Phases". Для этого необходимо:

- выбрать в каком target'е будет выполнятся скрипт (в нашем случае "Application" target);
- добавляем новый скрипт для выполнения: "Add Build Phase -> Add Run Script" и именуем его соответственно: "NIXObjCLint.py";
- в качестве выполняемого кода указываем данный скрипт, который имеет один параметр "--source" - директория с файлами для валидации;
выглядит следующим образом:

./Scripts/NIXObjCLint/NIXObjCLint.py --source="${SRCROOT}/Source"

  В случае, если указанный путь для валидации не существует, Xcode выдаст соответствующую ошибку и сборка будет приостановлена.
  Если все шаги проделаны правильно, то при нарушение правил форматирования кода Xcode выдаст соответствующее предупреждение и подсветит нужную строку, где была сделана ошибка.
  
  
  Для того, чтобы добавить валидацию файлов по заданному шаблону, необходимо задать свой собственный шаблон в файле и
  затем передать путь этого файла через аргумент "--template_file" в скрипт "NIXObjCLint.py". Шаблон задается регулярным выражением.
  Аргумент "--template_file" является необязательным и выглядит следующим образом:

./Scripts/NIXObjCLint/NIXObjCLint.py --source="${SRCROOT}/Source" --template_file="${SRCROOT}/Resources/MyFileTemplate"


  Чтобы добавить путь файла/файлов в исключение валидатора, необходимо создать файл, в котором указаны паттерны путей исключений.
  Каждый паттерн записывается в новой строке и задается маской в Unix shell-формате. Пример масок:

*.m
*_autogenerated.h
*/ExcludedDirectory*
...

   Путь файла, который содержит маски путей исключений, необходимо передать через аргумент "--excluded_files" в скрипт "NIXObjCLint.py".
   Аргумент является необязательным и выглядит следующим образом:

./Scripts/NIXObjCLint/NIXObjCLint.py --source="${SRCROOT}/Source" --excluded_files="${SRCROOT}/Resources/ExcludedFiles"
